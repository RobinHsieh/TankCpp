# Savage Tank Assault

Little fun game for MacOS.

## Project Structure
```
PROJECT_ROOT
├── assets/
│   ├── docs/
│   ├── imgs/
│   └── REFERENCE.md
├── main.cpp
├── README.md
└── LICENSE
```

## How to compile
```
gcc -o main.out main.cpp -lncurses
g++ -o main.out main.cpp -lncurses
```
